Install as App

🤖 Android (Chrome)
Tap the ⋮ menu (top-right) → select "Install app" or "Add to Home screen".

🍎 iOS / iPhone (Safari)
Tap the Share button (square with an up arrow) at the bottom → scroll down → select "Add to Home Screen".

💻 Desktop (Chrome / Edge)
Click the install icon (monitor with a down arrow) in the address bar. Or use the ⋮ menu → "Save and share" → "Install page as app".

Most modern browsers that support Progressive Web Apps (PWA) offer an install option — the exact wording may vary. If you don't see the options above, look for "Add to Home Screen", "Install", or a similar option in your browser's menu.

How to update

  1. Open your Google Sheet → Extensions → Apps Script.
  2. Select all existing code and replace it with the code below (use Copy).
  3. Click Deploy → Manage deployments.
  4. Click the ✏️ edit icon on your active deployment.
  5. Set Version to New version, then click Deploy.
  6. The Web App URL stays the same — no need to change it in Settings.
